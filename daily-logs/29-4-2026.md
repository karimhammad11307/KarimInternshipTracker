##  Product Strategy 

### 3. Shopify vs. Puck (System Design Comparison)
Conducted a comparative analysis between Shopify's visual builder ecosystem and our custom Puck implementation to identify what we need to build from scratch.

| Feature Area | Shopify Approach | Our Puck Architecture | Required Customization / Blockers |
| :--- | :--- | :--- | :--- |
| **Data Structure** | Proprietary Liquid schema | Pure JSON / BSON for MongoDB | We must design the Mongoose schemas to safely store and version the Puck JSON payloads. |
| **Component Library** | Massive marketplace of sections | Custom React Components | We need to build a specialized, hospitality-focused block library (Menu items, allergens, specials). |
| **Theming** | Global theme settings | CSS Variables & Puck Overrides | Need to map out how user-selected brand colors safely override our base Tailwind classes. |
| **SEO Strategy** | Server-Side Rendered (SSR) | TanStack Selective SSR | Need to strictly enforce `ssr: true` on the public storefront routes for Google indexing. |
